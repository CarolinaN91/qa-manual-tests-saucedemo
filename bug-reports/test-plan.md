Plano de testes - SauceDemo

Objetivo
Validar as principais funcionalidades de um e-commerce.

Escopo
- Login
- Produtos
- Carrinho
- Checkout

Tipos de teste
- Funcional
- Exploratório
- Usabilidade

Critérios de sucesso
- Usuário consegue completar compra sem erros críticos

Riscos
- Dependência de ambiente externo (site público)
- Possíveis inconsistências de estado da aplicação

Ambiente de teste
- Navegador: Chrome
- Sistema operacional: Windows
